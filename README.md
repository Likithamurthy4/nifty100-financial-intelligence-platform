# Nifty 100 Financial Intelligence Platform

## Sprint 1 – Data Foundation

### Objective
Build a robust ETL pipeline to load, validate, and store financial data of Nifty 100 companies into a SQLite database.

### Tech Stack
- Python
- Pandas
- SQLite
- SQLAlchemy
- Streamlit
- Plotly
- Pytest

### Sprint 1 Deliverables
- Data Loader
- Data Normalizer
- Data Validator
- SQLite Database
- Audit Reports
- Unit Tests
